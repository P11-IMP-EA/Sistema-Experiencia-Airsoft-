# Reglas de negocio

- Una reserva pertenece a un usuario y a un evento.
- Un usuario no puede reservar dos veces el mismo evento.
- Un evento tiene cupo total y cupo de alquiler.
- Los assets son unidades fisicas individuales.
- Un asset puede estar disponible, reservado, entregado, devuelto o en reparacion.
- La disponibilidad futura debe considerar asignaciones por evento.
- La operacion diaria debe poder hacerse desde celular o notebook.
- Toda consulta repetitiva debe convertirse en accion de UI.
